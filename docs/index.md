# Welcome to WgpArch Awesome WM

![Awesome WM Screenshot](screenshots/Launcher-menu_2026-06-23_16-40-15.png)

Highly configurable, dynamic window manager for X11 on Arch Linux.

## ✨ Features

- **Custom Rofi:** Themed power menu and launcher.
- **Train-Station Theme:** Custom Lua theme for a unique aesthetic.
- **Integrated Scripts:** Screenshot and weather utilities.

!!! tip
    Use the navigation menu on the left to view Installation, Keybinds, and Gallery.

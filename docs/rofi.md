Rofi Menus

Instead of the default Awesome WM menu, this setup utilizes Rofi for a more modern look.

     Launcher Menu: Triggered via your keybind, runs rofi/screenshot.sh to display open windows and apps.
     Power Menu: A beautifully styled exit menu (rofi/powermenu.sh) with options for Shutdown, Reboot, Lock, and Logout.
     Weather Script: Integrates rofi/weather.py to fetch and display weather data.
